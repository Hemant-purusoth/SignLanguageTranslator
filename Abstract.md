# Abstract

This project presents an interactive, web-based platform designed to bridge communication gaps for the deaf and hard-of-hearing community by integrating advanced real-time sign language translation into a secure video calling environment. The system empowers users to communicate naturally using sign language, which is actively translated to text in real-time for the recipient. By leveraging a robust and scalable architecture, the application separates a Node.js backend for secure WebRTC peer-to-peer signaling from a Flask-based Python backend dedicated solely to high-performance inference and customized machine learning training.

To overcome the challenges of classifying diverse sign language dialects and personal variations, the platform employs a specialized dual-model machine learning pipeline. For stationary alphabet and posture signs, a fast Random Forest model evaluates single-frame 2D spatial coordinates to recognize gestures instantly. For fluid, multi-step dynamic gestures, a Long Short-Term Memory (LSTM) neural network processes temporal windows of 30 continuous frames incorporating 3D depth data. At its core, the platform utilizes MediaPipe to efficiently extract high-fidelity 3D hand landmarks directly from live webcam feeds, minimizing latency while maximizing inference accuracy. This real-time computer vision processing seamlessly feeds into a secure, socket-based video conferencing application that enforces a strict two-person dynamic per unique room code, prioritizing privacy while actively relaying translated text to the remote participant's display.

Furthermore, the platform acknowledges the evolving nature of sign language through a personalized gesture training dashboard. Users can easily capture new static or dynamic signs by recording spatial sequences directly in their browser. The Python backend processes this novel data to provide single-click, background retraining of the custom classification models. By combining state-of-the-art temporal machine learning, personalized vocabulary expansion, and accessible real-time video communications, this project delivers a comprehensive ecosystem that not only translates actively but continuously adapts to its users, thereby fostering an inclusive and global digital communication experience.
